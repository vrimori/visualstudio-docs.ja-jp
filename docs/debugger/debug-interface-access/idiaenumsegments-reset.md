# <a name="idiaenumsegmentsreset"></a>IDiaEnumSegments::Reset
先頭に、列挙体シーケンスをリセットします。  
  
## <a name="syntax"></a>構文  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>戻り値  
 `S_OK` を返します。
