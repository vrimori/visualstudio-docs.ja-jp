# <a name="c6317"></a>C6317
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このトピックの最新バージョンをご覧[C6317](https://docs.microsoft.com/visualstudio/code-quality/c6317)します。  
  
警告 C6317: 不適切な演算子です: 論理否定演算子は 1 の補数演算子 (~) と代替できません  
  
 この警告が示す論理 not (`!`) ビット フラグを使用する可能性がある定数に適用されています。 論理否定の結果はブール値です。ビット演算を適用する位置が正しくない- と (`&`) 演算子をブール値。 1 の補数演算子を使用して (`~`) フラグを反転する演算子。  
  
## <a name="example"></a>例  
 この警告が発生するコード例を次に示します。  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & !FLAGS) // warning  
  {   
    // code  
  }  
}  
```  
  
 この警告を解決するには、次のコードを使用します。  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & ~FLAGS)  
  {  
    // code  
  }  
}  
```
