# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28279: シンボルについて、'begin' が見つかりましたが対応する 'end' がありません  
  
 Begin と end 注釈言語をサポートしています (`{`と`}`C で) 構造とペアリングされている負荷が不均衡です。 マクロを使用する場合、このような状況を回避できます。
