# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28289: 関数: ReadableTo または WritableTo にパラメーターとして limit-spec がありません  
  
 コード分析ツールは、ときに関数/パラメーターの注釈は誤った説明したように、この警告を報告します。
