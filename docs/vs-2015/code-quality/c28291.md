# <a name="c28291"></a>C28291
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28291: Post で null/notnull deref レベル 0 は関数に対して意味のない\<x > パラメーターで\<番号 >  
  
 コード分析ツールでは、null および非 null プロパティを逆参照レベル 0 のパラメーターの事後条件を指定すると、この警告を報告します。 レベル 0 は変更できませんに値を逆参照するため、このエラーが発生します。
