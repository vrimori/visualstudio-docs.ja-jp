# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28236: 注釈は式内で使用できません  
  
 この警告は、式のコンテキストで注釈として宣言された関数を使用しようとすることを示します。 注釈は標準のマクロが使用されている場合があります。この警告は、一般的な使用で目にする必要はありません。
