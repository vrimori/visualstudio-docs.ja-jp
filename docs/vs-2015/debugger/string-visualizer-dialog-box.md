# <a name="string-visualizer-dialog-box"></a>[String ビジュアライザー] ダイアログ ボックス
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

このダイアログ ボックスは、Visual Studio デバッガーに含まれる標準のビジュアライザーの 1 つです。 String ビジュアライザーは、通常のテキストの他にも HTML 文字列と XML 文字列を評価し、その結果をわかりやすい形式で表示できます。
