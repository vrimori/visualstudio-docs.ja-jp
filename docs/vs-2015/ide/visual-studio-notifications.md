# <a name="visual-studio-notifications"></a>Visual Studio の通知
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio には、さまざまなイベントを警告する通知システムがあります。

1. Visual Studio 内のコンポーネントに対して使用できる更新プログラムがあります

2. Visual Studio 自体に使用可能な更新プログラムがあります

3. 関心があるかもしれない環境イベントがありました

   Visual Studio は、Visual Studio 自体またはインストールされているいずれかのコンポーネントの更新プログラムが入手可能な場合に通知します。 また、Visual Studio 環境で発生する特定の種類のイベントに関する通知も提供します。 通知アイコンは、Visual Studio のタイトル バーの右上にあるフラグです。アイコンが赤の場合は、ライセンスに問題があることを意味します。 黄色の場合は、インストールできる Visual Studio 製品の更新プログラムが存在します。 「標準」または黒の場合は、情報提供またはオプションのメッセージ (回避策を見つける必要があるクラッシュなど) が表示されます。 アイコンをクリックすると、**通知ハブ**が開き、操作対象とする通知を選択できます。 (通知の上にマウス ポインターを置いて) 通知を 1 回限りで破棄するか、通知を常に無視するかを選択できます。後者の場合、Visual Studio にそのメッセージは表示されなくなります。

   通知を常に無視することを選択した場合、Visual Studio によって表示されなくなります。 [[通知]、[環境]、[オプション]](../ide/reference/notifications-environment-options-dialog-box.md) ダイアログ ボックスで、無視される通知のリストをリセットできます。
