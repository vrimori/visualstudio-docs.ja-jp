# <a name="copy-selection"></a>選択範囲のコピー
レポート タブから呼び出し履歴全体をコピーするには、**[コピー]** をクリックします。 その後、貼り付けをサポートする任意のプログラムに呼び出し履歴を貼り付けることができます。
