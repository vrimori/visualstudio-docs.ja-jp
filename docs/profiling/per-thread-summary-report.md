# <a name="per-thread-summary-report"></a>スレッド別の概要レポート
この棒グラフは、現在の表示時間範囲におけるアクティビティ カテゴリごとに、非表示にされていない各スレッドで費やされた時間の割合を示します。 "実行" はそのスレッドが実行中であることを示し、他のすべてのカテゴリはそのスレッドが何らかの処理の待機中であることを示します。
