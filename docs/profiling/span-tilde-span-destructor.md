# <a name="spanspan-destructor"></a>span::~span デストラクター
`span` オブジェクトを破棄し、そのリソースを解放します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
~span();  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** *cvmarkersobj.h*  
  
 **名前空間:** Concurrency::diagnostic
